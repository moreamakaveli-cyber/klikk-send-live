# Hently

Nettside for Hently (Next.js App Router).

## Utvikling

```bash
npm install
npm run dev
```

Åpne [http://127.0.0.1:3000](http://127.0.0.1:3000). Får du **404 på forsiden**, kjør `npm run dev:clean` (sletter `.next` og starter på nytt). Ved Turbopack-problemer: `npm run dev:turbo` etter `rm -rf .next`.

## Produksjon

```bash
npm run build
npm start
```
